User Registration


URI       /login
Method	  POST
Data	  {
          "act" : "signup",
          "ac"  : "user1",
          "pw"  : "pass1"
}
Result	{
     "usernamedisplay": "user1",
     "loginstatus": "signup" 
}
Error	{
    "usernamedisplay": "user1",
    "loginstatus": "duplicate" //when username is already created
}
